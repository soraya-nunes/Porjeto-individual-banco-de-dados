Projeto individual - Módulo 4

Existem outras entidades além dessas três? 
R: Sim, em uma instituição de ensino existem várias como instrutores, matrículas e ect.

Quais são os principais campos e tipos? 
R: Na tabela alunos foi INT para o primeiro aluno e VARCHAR para o resto, na tabela turmas foi INT para a primeira turma e VARCHAR para o resto e na tabela cursos foi INT para a primeira turma e VARCHAR para o resto.

Como essas entidades estão relacionadas? 
R: É um curso para cada tipo de turma, vários professores para varias turmas, vários alunos para várias turmas e vários professores para vários alunos.
